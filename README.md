# Usborne Coding Books in Python

I grew up reading the Usborne computer coding books. Typing out the BASIC code, playing the games, and hacking in my own changes. 

I am recreating the games in Python, staying faithful to the original gameplay and text-based charm, but rewriting the underlying code using a modern programming style.

The original books are freely available as PDFs from the Usborne website:
[Usborne Computer and Coding Books](https://usborne.com/za/books/computer-and-coding-books)

The original BASIC code was constrained by the hardware of the era (like the Commodore VIC-20 or ZX Spectrum), which led to single-character variables, `GOTO` statements, hardcoded magic numbers, and all-caps text. It's been a fun challenge to interpret and rewrite.
